# How to build a House Model
<HR>

<H3>There are 3 type of house you can build</H3>
1. Big House Model
![pic_90](images/assembly/house_big_topic.png)
2. Small House Model
![pic_90](images/assembly/house_small_topic.png)
3. Open Style Model
![pic_90](images/assembly/house_open_topic.png)

<P>
To build a model, the house base is model A to model C.<P>

## Big House Model

### Part list
![pic_80](images/assembly/house_big_parts.png)

### Assembly

<span id="subtitle">Step 1</span><BR><P>
* Add the 6 rubber bends to the bottom of the model B4 and B2.<P>

![auto_fit](images/assembly/house_big_ass0_new1.png)

<span id="subtitle">Step 2</span><BR><P>
* Insert the model B2 and B4 to model A.<P>

![auto_fit](images/assembly/house_big_ass1_new1.png)


<span id="subtitle">Step 3</span><BR><P>
* Add the 6 rubber bends to the bottom of the model B1 and B3.<P>

![auto_fit](images/assembly/house_big_ass1_5_new1.png)

<span id="subtitle">Step 4</span><BR><P>
* Put the model B1, B3 to model A, and then insert to the model B2 and B4.<P>

![auto_fit](images/assembly/house_big_ass2_new1.png)

<span id="subtitle">Step 5</span><BR><P>
* Make sure <u>all the holes</u> of model A wrap the rubber from the model B1 to B4 properly and completely.<P>

![auto_fit](images/assembly/house_big_ass2_5_new1.png)

<span id="subtitle">Step 6</span><BR><P>
* Add the 4 rubber bends to the front side (Model B1) and back side (Model B3) of the house.<P>

![auto_fit](images/assembly/house_big_ass3_new1.png)

<span id="subtitle">Step 7</span><BR><P>
* Insert the model C1 and C2 on model B1, B2, B3, B4, align with the holes on them.<P>

![auto_fit](images/assembly/house_big_ass4_new1.png)

<span id="subtitle">Step 8</span><BR><P>
* Insert the model C3 on the model C2.<P>

![auto_fit](images/assembly/house_big_ass5_new1.png)

<span id="subtitle">Step 9</span><BR><P>
* Completed!<P>

![auto_fit](images/assembly/house_big_ass6_new1.png)


### Optional: Extra Room
<HR>

<span id="subtitle">Step 1</span><BR><P>
* For extra room installation, Insert the model D1, D3 on model B1 align with<BR> the holes on model A.<P>

![auto_fit](images/assembly/house_extra_ass1_new1.png)

<span id="subtitle">Step 2</span><BR><P>
*  Put the model D2 on the holes of model A, and then insert it to model D1 and D3. <P>

![auto_fit](images/assembly/house_extra_ass2_new1.png)

<span id="subtitle">Step 3</span><BR><P>
*   Add the 4 rubber bends to the model D2 so that the house will not loose.<P>

![auto_fit](images/assembly/house_extra_ass2_5_new1.png)

<span id="subtitle">Step 4</span><BR><P>
*   Insert the model D4 to the model B1, and then attach it to the model D1, D2 and D3.
<P>

![auto_fit](images/assembly/house_extra_ass3_new1.png)

<span id="subtitle">Step 5</span><BR><P>
* Completed! <P>

![auto_fit](images/assembly/house_extra_ass4_new1.png)

## Small House Model

Before we build a small house model, we need to cut the cardboard as instruction on cardboard.

### Part list

![pic_80](images/assembly/house_small_parts.png)

### Cut the cardboard


<span id="subtitle">Step 1</span><BR><P>
* Cutting with the dashed line on A cardboard. <P>

![auto_fit](images/assembly/house_small_cut1_new.png)


<span id="subtitle">Step 2</span><BR><P>
* Cutting with the dashed line on C1 and C2 cardboard.<P>

![auto_fit](images/assembly/house_small_cut2_new.png)


### Assembly

<span id="subtitle">Step 1</span><BR><P>
* Add the 6 rubber bends to the bottom of the model B5 and B6. <P>

![pic_100](images/assembly/house_small_ass1_new.png)

<span id="subtitle">Step 2</span><BR><P>
* Insert the model B5 and B6 to model A. <P>

![pic_100](images/assembly/house_small_ass2_new.png)

<span id="subtitle">Step 3</span><BR><P>
* Put the model B3 to model A, and then insert to the model B5 and B6. put the model B1 on the model A, and then insert it to the model B5 and B6. <P>

![pic_100](images/assembly/house_small_ass3_new.png)

<span id="subtitle">Step 4</span><BR><P>
* Make sure <u>all the holes</u> of model A wrap the rubber from the model B1 to B4 properly and completely.<P>

![pic_100](images/assembly/house_small_ass3_5_new.png)

<span id="subtitle">Step 5</span><BR><P>
* Add the 8 rubber bends to the model B1 and B3 so that the house will not loose. <P>

![pic_100](images/assembly/house_small_ass4_new.png)

<span id="subtitle">Step 6</span><BR><P>
*  Insert the model C1 and C2 on model B1, B5, B3, B6, align with the holes on them.
![pic_100](images/assembly/house_small_ass5_new.png)


<span id="subtitle">Step 7</span><BR><P>
*  Insert model C3 on the mode C2.
![pic_100](images/assembly/house_small_ass6_new.png)

<span id="subtitle">Step 8</span><BR><P>
*  Completed!
![pic_100](images/assembly/house_small_ass7_new.png)
